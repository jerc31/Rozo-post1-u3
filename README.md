## Descripción del proyecto

Este proyecto corresponde al laboratorio **"Página de Perfil con CSS3"** de la Unidad 3.

El objetivo del laboratorio es aplicar conceptos fundamentales de **CSS3**, incluyendo:

- Box Model y `box-sizing`
- Selectores avanzados
- Pseudo-clases y pseudo-elementos
- Variables CSS (Custom Properties)
- Posicionamiento (`fixed`, `relative`, `absolute`)
- Estilado de formularios accesibles

La página implementa un **perfil personal con tres secciones principales**:

- Perfil del estudiante
- Lista de habilidades
- Formulario de contacto

---

## Tecnologías utilizadas

- HTML5 semántico
- CSS3
- Flexbox
- Custom Properties (Variables CSS)
- Selectores avanzados
